# 전체 테스트 품질 검증

백엔드, 프론트엔드, E2E 테스트 품질을 **병렬로** 검증합니다.

---

## 수행 방법

```
3개의 Task를 병렬로 실행:

1. Task (subagent) → /verify (백엔드)
2. Task (subagent) → /verify-fe (프론트엔드)
3. Task (subagent) → /verify-e2e (E2E)
```

---

## 실행 지시

아래 3개 Task를 **반드시 병렬로** (단일 메시지에서 동시에) 실행할 것:

### Task 1: 백엔드 테스트 검증
```
.claude/commands/verify.md 규칙에 따라 백엔드 테스트 품질 검증 수행.
변경된 server/**, tests/** 파일의 관련 테스트를 분석하고 리포트 출력.
```

### Task 2: 프론트엔드 테스트 검증
```
.claude/commands/verify-fe.md 규칙에 따라 프론트엔드 테스트 품질 검증 수행.
변경된 components/** 파일의 관련 테스트를 분석하고 리포트 출력.
```

### Task 3: E2E 테스트 검증
```
.claude/commands/verify-e2e.md 규칙에 따라 E2E 테스트 품질 검증 수행.
변경된 페이지/기능의 관련 E2E 테스트를 분석하고 리포트 출력.
```
