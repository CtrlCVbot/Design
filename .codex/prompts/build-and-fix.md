# 빌드 및 에러 수정

TypeScript 빌드를 실행하고 발견된 에러를 수정합니다.

## 수행 단계

1. **빌드 실행**: `npm run build` 또는 `npx tsc --noEmit`
2. **에러 분석**:
   - 에러 개수 확인
   - 에러 유형별 분류
   - 파일별 그룹화
3. **수정 전략 결정**:
   - 5개 미만: 순차적으로 직접 수정
   - 5개 이상: 우선순위 정하고 순차 수정
4. **수정 후 재빌드**: 에러 0개 될 때까지 반복

## 에러 우선순위

1. **Critical**: import 에러, 모듈 찾기 실패
2. **High**: 타입 불일치, 필수 속성 누락
3. **Medium**: 암시적 any, 사용되지 않는 변수
4. **Low**: 스타일 관련

## 출력 형식

```
## 빌드 결과

### 발견된 에러: N개

#### 파일별 분류
- path/to/file1.ts: 3개
- path/to/file2.ts: 2개

#### 수정 계획
1. [ ] file1.ts:23 - TS2322 타입 불일치
2. [ ] file1.ts:45 - TS2345 인자 타입
...

### 수정 진행
- [x] file1.ts:23 수정 완료
- [ ] file1.ts:45 수정 중...
```

## 주의사항
- 수정 후 반드시 재빌드하여 확인
- 연쇄적으로 발생하는 에러 주의
- 한 번에 너무 많이 수정하지 말고 단계적으로
