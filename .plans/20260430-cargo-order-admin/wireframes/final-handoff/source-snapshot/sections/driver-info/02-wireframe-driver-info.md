├──────────────────────────────────────────────────────────────────────────────┤
│ 차주 │ 차주명   │ 차량번호        │ 차주 연락처      │ 톤수 │ 차종 │ 변경 │

## 2. 입력 전/조회

```text
┌──────────────────────────────────────────────────────────────┐
│ 차주 │ .................................... │ 차주 정보 입력 │
└──────────────────────────────────────────────────────────────┘
```

CTA 클릭 후:

```text
차주 정보 입력 버튼이 왼쪽 `차주` 라벨 방향으로 흡수
→ 차주/차량 통합 조회 다이얼로그 오픈
→ 행 선택
→ 차주 정보에 적용
```

## 3. 조회 다이얼로그

```text
┌──────────────────────────────────────────────────────────────────────────────┐
│ 차주/차량 통합 조회                                                   닫기 │
├──────────────────────────────────────────────────────────────────────────────┤
│ [내부 DB] 검색 전에는 결과 목록을 숨김                       [내부 DB 조회] │
│ [차주명, 차량번호, 연락처 검색                 ] [조회] [검색 초기화]      │
│ [통합검색] [차주명] [차량번호] [연락처] [톤수] [차종]                     │
├──────────────────────────────────────────────────────────────────────────────┤
│ 검색 전: 차주를 먼저 조회하세요 / [차주 등록]                              │
├──────────────────────────────────────────────────────────────────────────────┤
│ 조회 후:                                                    [차주 등록]     │
│ 선택 │ 차주명 │ 차량번호 │ 차주 연락처 │ 톤수 │ 차종 │ 최근 운행             │
│  ○   │ 임동진 │ 경기92...│ 010-...     │ 5톤  │ 축차 │ 완료 12건             │
│  ○   │ 박운송 │ 경기80...│ 010-...     │ 17톤 │ 플러스윙 │ 최근 배차 가능       │
├──────────────────────────────────────┬───────────────────────────────────────┤
│ 결과 목록                            │ 안내: 행 선택 또는 차주 등록            │
│                                      │ 선택 시: 선택 미리보기                 │
│                                      │ 등록 클릭 시: 차주 등록 폼             │
└──────────────────────────────────────┴───────────────────────────────────────┘
```

오른쪽 패널은 한 번에 하나의 상태만 표시합니다.

| 상태 | 표시 |
| --- | --- |
| 검색 전 | 결과 목록 숨김, 중앙 `차주 등록` CTA |
| 내부 DB 조회/검색 후 | 결과 목록 표시, 리스트 우측 상단 플로팅 `차주 등록` CTA |
| 조회 행 선택 | 선택 미리보기 |
| 차주 등록 클릭 | 차주명, 차량번호, 연락처, 톤수, 차종 등록 폼 |
| 등록 완료 | 조회 목록에 신규 행 추가 후 자동 선택 |

## 4. 수정 기준

| 필드 | 수정 방식 |
| --- | --- |
| 차주명 | `차주/차량 변경`으로 재조회 |
| 차량번호 | `차주/차량 변경`으로 재조회 |
| 차주 연락처 | 적용 후 row에서 inline edit |
| 톤수 | 적용 후 row에서 select 변경 |
| 차종 | 적용 후 row에서 select 변경 |

## 5. 변경 애니메이션

```text
[차주/차량 변경] 클릭
→ 버튼이 왼쪽 차주 row 방향으로 흡수
→ row가 부드럽게 highlight
→ 조회 다이얼로그 오픈
```

## 6. B 통합 기준

| 기준 | 값 |
| --- | --- |
| section title | `5. 차주 정보` |
| row label | `차주` |
| action label | `차주/차량 변경` |
| dialog title | `차주/차량 통합 조회` |
| inline edit | 차주 연락처 input, 톤수/차종 select |
| dialog CTA | 검색 전 중앙 CTA, 조회 후 플로팅 CTA |
| dialog clarity | 고정 컬럼 폭, 정수 line-height/padding으로 하단 리스트 흐림 방지 |
