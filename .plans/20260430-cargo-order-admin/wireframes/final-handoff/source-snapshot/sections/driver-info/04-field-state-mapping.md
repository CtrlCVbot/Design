# Field State Mapping - 차주 정보

## 필드 매핑

| 필드 | 표시 | 변경 방식 | 임시 수정 | 비고 |
| --- | --- | --- | --- | --- |
| 차주명 | row value | 조회 다이얼로그 | 불가 | 배차 수행자 식별 |
| 차량번호 | row value | 조회 다이얼로그 | 불가 | 등록 차량 식별값 |
| 차주 연락처 | row value | 조회 다이얼로그 | 가능 | 현재 오더 기준 보정 |
| 톤수 | row value | 조회 다이얼로그 | 가능 | 현재 오더 기준 스펙 보정 |
| 차종 | row value | 조회 다이얼로그 | 가능 | 현재 오더 기준 스펙 보정 |
| 차주/차량 변경 | button | 다이얼로그 오픈 | 해당 없음 | A안 흡수 전환 |

## 상태 값

| 상태 | 표시 | action |
| --- | --- | --- |
| Ready | 5개 필드 row | 차주/차량 변경 |
| Empty | 빈 row + CTA | 차주 정보 입력 |
| Dialog Open | 검색 전 안내 + 중앙 차주 등록 CTA | 내부 DB 조회, 검색, 차주 등록 |
| Dialog Results | 결과 목록 + 플로팅 차주 등록 CTA | 행 선택 또는 신규 등록 |
| Preview | 결과 목록 + 선택 미리보기 | 차주 정보에 적용 |
| Register | 결과 목록 + 차주 등록 폼 | 신규 차주 등록 |
| Editing | 연락처 input + 톤수/차종 select | blur, Enter 또는 change 저장 |

## 다이얼로그 표시 품질 기준

| 항목 | 기준 |
| --- | --- |
| 결과 컬럼 | `선택 / 차주명 / 차량번호 / 연락처 / 톤수 / 차종 / 최근 운행 또는 상태` 고정 폭 |
| 결과 행 | 정수 `padding`과 `line-height`로 텍스트 반 픽셀 렌더링 방지 |
| 선택 마커 | 12px 원형 + 정수 margin 기준 |
| CTA 표시 | 검색 전 중앙 CTA, 조회 후 플로팅 CTA, 등록 폼 열림 시 CTA 숨김 |

## 접근성 기준

| 항목 | 기준 |
| --- | --- |
| 조회 결과 행 | `role="button"`, `tabindex="0"` |
| 적용 버튼 | 선택 전 disabled |
| inline edit | focus/Enter로 input 전환 |
| 다이얼로그 | `role="dialog"`, `aria-modal="true"` |
