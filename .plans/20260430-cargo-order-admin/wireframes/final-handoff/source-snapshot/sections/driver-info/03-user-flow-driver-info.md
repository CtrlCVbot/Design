# User Flow - 차주 정보

## 1. 값 있음 확인

```mermaid
flowchart TD
  A["차주 정보 row 표시"] --> B["차주명/차량번호/연락처/톤수/차종 확인"]
  B --> C{"연락처, 톤수, 차종 보정 필요?"}
  C -->|아니오| D["다음 섹션 이동"]
  C -->|예| E["해당 값 클릭"]
  E --> F["연락처 input 또는 톤수/차종 select 전환"]
  F --> G["저장 후 row 복귀"]
```

## 2. 차주/차량 변경

```mermaid
flowchart TD
  A["차주/차량 변경 클릭"] --> B["버튼 흡수 애니메이션"]
  B --> C["통합 조회 다이얼로그 오픈"]
  C --> D["검색 전 안내 + 중앙 차주 등록 CTA"]
  D --> E["내부 DB 조회 또는 검색 실행"]
  E --> F["결과 목록 + 플로팅 차주 등록 CTA"]
  F --> G["결과 행 선택"]
  G --> H["선택 미리보기 확인"]
  H --> I["차주 정보에 적용"]
  I --> J["row 갱신"]
```

## 3. 입력 전 상태

| 단계 | 화면 반응 |
| --- | --- |
| 차주 정보 없음 | 빈 row와 `차주 정보 입력` CTA 표시 |
| CTA 클릭 | 버튼이 `차주` 라벨 방향으로 흡수 |
| 조회 다이얼로그 | 검색 전 안내 표시, `내부 DB 조회` 또는 검색 후 목록 표시 |
| 적용 | 값 있음 row로 전환 |

## 4. 차주 등록

```mermaid
flowchart TD
  A["검색 전 중앙 CTA 또는 검색 후 플로팅 차주 등록 클릭"] --> B["선택 미리보기 숨김"]
  B --> C["차주 등록 폼 표시"]
  C --> D["차주명/차량번호/연락처/톤수/차종 입력"]
  D --> E["차주 등록"]
  E --> F["결과 목록에 신규 차주 행 추가"]
  F --> G["신규 행 자동 선택"]
  G --> H["선택 미리보기 노출"]
```

## 5. 보류 흐름

| 흐름 | 보류 사유 |
| --- | --- |
| 차량번호 검증 실패 | 실제 검증 규칙 미정 |
| 차주 마스터 저장 | 저장 위치 미정 |
| 톤수/차종 비교 | 화물 운송정보 요구 조건과 차주 차량 스펙 비교 방식 미정 |
