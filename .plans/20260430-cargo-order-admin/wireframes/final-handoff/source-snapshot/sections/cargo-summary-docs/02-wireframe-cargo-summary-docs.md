# Wireframe - 화물정보 요약

## 1. 기본 구조

```text
┌────────────────────────────────────────────────────────────────────────────┐
│ 3 화물정보 요약                         화물 운송정보에서 생성되는 요약 row │
├────────────────────────────────────────────────────────────────────────────┤
│ ┌────────────────────────────────────────────────────────────────────────┐ │
│ │ 요약 │ 화물정보 요약                                      │ 수정      │ │
│ │      │ 지금상 당착 장비 운송 지/지-A. 판넬 12파렛트...    │           │ │
│ └────────────────────────────────────────────────────────────────────────┘ │
└────────────────────────────────────────────────────────────────────────────┘
```

## 2. 상태별 구조

### A. 값 있음

```text
요약 | 화물정보 요약
     | 지금상 당착 장비 운송 지/지-A. 판넬 12파렛트, 5.5톤 축차, 상하차 지게차 조건.
                                                    [수정]
```

### B. 값 없음

```text
요약 | 화물정보 요약
     | 화물 운송정보 입력 후 요약 생성
                                                    [요약 대기]
```

### C. 수정 중

```text
요약 | 화물정보 요약
     | [지금상 당착 장비 운송 지/지-A. 판넬 12파렛트...                  ]
                                                    [저장]
```

## 3. 배치 기준

| 영역 | 기준 |
| --- | --- |
| row wrapper | `summary-card` |
| row | `summary-row` |
| label chip | `kind` |
| 값 셀 | `cell`, `cell-label`, `cell-value` |
| action | `row-action` |
| 최소 폭 | 720px |
| row padding | `5px 8px` |
| gap | `7px` |

## 4. Responsive 기준

| 화면 폭 | 처리 |
| --- | --- |
| 880px 이상 | 한 줄 row 유지 |
| 879px 이하 | row 내부는 최소 폭을 유지하고 card에서 가로 스크롤 허용 |

## 5. 삭제 기준

| 삭제 항목 | 처리 |
| --- | --- |
| 중량 | 화면에서 제거 |
| 계산 기준 | 화면에서 제거 |
| 이하 여부 | 화면에서 제거 |
| 안내문 | 화면에서 제거 |
| 증빙 옵션 | 화면에서 제거 |
