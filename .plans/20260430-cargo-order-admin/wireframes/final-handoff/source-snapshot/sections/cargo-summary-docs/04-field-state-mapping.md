# Field State Mapping - 화물정보 요약

## 필드 매핑

| 필드 | 출처 | 기본 상태 | 상호작용 | 비고 |
| --- | --- | --- | --- | --- |
| 화물정보 요약 | `3. 화물 운송정보`와 `2. 운송 구간` 파생 | 표시 | 수정 버튼 또는 값 선택 | 긴 문장은 말줄임 |
| 자동 생성값 | `품목`, `실중량`, `톤수`, `차종`, `대수`, 상하차 일시/방법 | `auto` | 원본 변경 시 재생성 | 금액 조건은 기본 제외 |
| 사용자 override | 사용자가 요약 row에서 직접 수정 | `overridden` | 같은 위치 input 전환 | 원본 필드는 변경하지 않음 |
| stale 후보 | 원본 변경 후 override와 불일치 가능 | `stale` | 재생성 또는 수정 유지 검토 | chip 노출 여부는 후속 결정 |
| 수정 | 사용자 action | 버튼 | input 전환 | 값 없음 상태에서는 대기 표시 |

## 상태 값

| 상태 | 요약 값 | action | 표시 |
| --- | --- | --- | --- |
| Auto | 파생 요약 문장 | 수정 | 손글씨형 값 |
| Overridden | 사용자가 보정한 문장 | 수정 | 손글씨형 값. 원본 자동 갱신은 보류 |
| Stale | 원본과 다를 수 있는 override | 재생성 또는 수정 유지 | chip 노출 여부는 후속 결정 |
| Empty | `화물 운송정보 입력 후 요약 생성` | 요약 대기 | muted placeholder |
| Editing | input value | 저장 | 같은 row 안에서 입력 |

## 원본 필드 연결

| 원본 필드 | 출처 | 요약 반영 |
| --- | --- | --- |
| `품목` | `3. 화물 운송정보`의 `운송+품목` 2번째 row | 문장의 핵심 물품명 |
| `실중량` | `3. 화물 운송정보`의 `실중량` | `5.5톤`처럼 실제 중량 표현 |
| `톤수` | `3. 화물 운송정보`의 `톤수` | 실중량 누락 시 대체 또는 차량 조건 보조 |
| `차종` | `3. 화물 운송정보`의 `차종` | `축차`, `윙바디` 같은 차량 조건 |
| `대수` | `3. 화물 운송정보`의 `대수` | 2대 이상일 때만 포함 |
| 상차/하차 일시 | `2. 운송 구간` | `지금상`, `당착` 같은 짧은 조건 |
| 상하차 방법 | `2. 운송 구간` | `상하차 지게차 조건` 같은 작업 조건 |

금액 조건(`결제방법`, `청구비용`, `운송비용`, `수수료`, `수익/차주운임`)은 정산 정보이므로 요약 문장에 기본 포함하지 않습니다.

## 갱신 트리거

| 변경 이벤트 | Auto 상태 | Overridden 상태 |
| --- | --- | --- |
| `품목`, `실중량`, `톤수`, `차종` 변경 | 즉시 요약 재생성 | `stale` 후보 |
| `대수` 변경 | 2대 이상이면 요약 재생성 | `stale` 후보 |
| 상하차 일시/방법 변경 | 요약 조건 재생성 | `stale` 후보 |
| 금액 조건 변경 | 영향 없음 | 영향 없음 |

## 접근성 기준

| 항목 | 기준 |
| --- | --- |
| 요약 값 | `tabindex="0"`로 focus 가능 |
| 수정 버튼 | native button 사용 |
| 수정 input | `aria-label="화물정보 요약 수정"` 제공 |
| focus | 운송구간/화주 정보와 같은 dashed focus 표시 |
