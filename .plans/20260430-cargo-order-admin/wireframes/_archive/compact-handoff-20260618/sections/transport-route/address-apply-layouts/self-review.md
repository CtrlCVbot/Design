## 결론

`transport-route-address-apply-layouts.html`은 1열 단일 행과 화주 주소록 + Kakao 통합 검색 기준으로 재정리되었다.

현재 기준은 상차/하차 각각 한 줄에 지명, 주소, 상세주소, 담당자, 연락처, 일시, 방법, 변경 버튼을 표시하는 것이다. 주소 검색 다이얼로그에서는 화주 등록 주소와 Kakao 검색 결과를 함께 보여준다. 주소는 고정하고, Kakao 결과는 지명 또는 장소명이 있을 때 저장 가능하며, 화주 등록 결과는 상세정보 변경이 있을 때 수정 저장 가능하다.
