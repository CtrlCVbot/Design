master 적용 전 비교판

현재 `⊟ 라벨: 표시` 버튼을 대체할 수 있는 UI를 같은 헤더 환경에서 시각화했습니다. 각 안의 `상태 전환` 버튼을 누르면 라벨 표시/숨김 상황도 함께 확인할 수 있습니다.

옵션 헤더 표현 장점 주의점 추천
1. 현재안 개선 라벨: 표시 / 숨김 기존 동작과 가장 가까움 라벨이라는 내부 용어가 남음 보류
2. 보기 옵션 보기 옵션 ▾ 나중에 설정이 늘어나도 안전함 한 번 더 열어야 함 추천
3. 폼 보기 모드 보기 [상세] [간편] 운영자가 의미를 바로 이해함 헤더 폭을 조금 차지함 추천
4. 단일 모드 버튼 상세 보기 / 간편 보기 간단하고 즉시 전환 가능 현재 상태와 다음 행동이 헷갈릴 수 있음 중간
5. 상태 chip 흡수 상태 chip 옆 `상세` 헤더가 가장 조용함 클릭 가능한지 약할 수 있음 중간
6. 아이콘 버튼 눈 아이콘 + tooltip 가장 미니멀함 처음 보는 사용자는 의미를 놓칠 수 있음 낮음
7. 스위치 라벨 표시 switch ON/OFF 상태가 명확함 설정 UI 느낌이 강함 중간
