## 핵심 요약

- **콕핏형 TMS**: 페이지 이동 없이 좌측 패널(운행등록 ↔ 차량/차주 배정 ↔ 대화방) × 우측 패널(운행내역 ↔ 추천목록 ↔ 접힘) 모드 전환으로 등록→배정→관제→소통 전 과업 처리.
- **3단계 AI 보조**: 리스트의 배차 난이도 신호등 → 확장 행의 AI 배차 분석 어드바이저(3단계) → 배정 패널의 AI 최적 추천 차주(사유 태그 포함).
- **입력 가속 체인**: 거래처 칩 → 자주 쓰는 구간 → 주소·운임(최근/최빈) 추천이 연쇄적으로 다음 입력을 좁혀줌.
- **기술**: Vite SPA, API 없음(전 데이터 클라이언트 목업), 외부 연동은 카카오 우편번호 1종. 디자인은 토스 계열(Pretendard, #3182F6).
