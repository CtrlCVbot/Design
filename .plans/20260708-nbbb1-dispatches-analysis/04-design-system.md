# nbbb1.vercel.app/dispatches — Design System 현황 문서

분석일: 2026-07-08 · **실제 배포 페이지에서 추출한 as-is 문서** (제안 아님)
추출 방법: 렌더 DOM에서 CSS 커스텀 프로퍼티·computed style 전수 수집

전반적으로 **토스(Toss) 디자인 언어 계열**: Pretendard 서체, #3182F6 프라이머리, #F2F4F6 회색 배경, 큰 라운드(12~18px), 보더 대신 배경색·그림자로 위계 표현.

## 2. 컴포넌트 인벤토리 (실측 스타일)

### 2-1. 버튼

| 종류 | 스타일 | 예 |
|---|---|---|
| Primary | bg `#3182F6` / 흰 텍스트 / r12 / h42 / w600 / shadow-sm | `+ 배차 등록`, `차량배정`, `수정`, `전송` |
| Secondary | bg `#E5E8EB` / `#191F28` / r12 / h42 | `초기화`, `검색`, `닫기` |
| Outline pill (활성) | bg `#E8F3FF` / text·border `#3182F6` / r-full | 활성 필터 칩 |
| Ghost pill (비활성) | bg `#F2F4F6` / text `#4E5968` / border `#E5E8EB` / r-full | 비활성 필터 칩 |
| 스테퍼 증액/감액 | 증액 빨강 계열·감액 파랑/회색 계열 소형 pill | `+1만` `-5천` 등 |
| 상태 버튼(선택됨) | 해당 상태 시맨틱 색으로 채움 (배차대기=주황, 운행완료=청록) | 상태 제어 그룹 |

### 2-2. 칩 (추천/입력 보조)

- bg `#E8F3FF` / text `#3182F6` / r-full / 11.5px w600 / padding 3×10px
- 종류: 최근 거래처, 자주 쓰는 구간, 지역, 차량 스펙, 화물품목, 날짜/정산 퀵버튼(사각 r8 변형)

### 2-3. 상태 뱃지 (r-full, 12px w600, padding 5×12)

| 상태 | 배경 | 텍스트 |
|---|---|---|
| 배차중 | `#FFF5E6` | `#D97706` |
| 배차완료 | `#E8F3FF` | `#3182F6` |
| 상차완료 / 하차완료 / 운행완료 | `#E6F9F7` | `#0F9F90` |
| 배차취소 | `#FEEBEE` | `#DC2626` |

+ 배차중 전용: 난이도 신호등 3점(7px 원, title 툴팁), 활성 점만 채색.

### 2-4. 입력 필드

- bg `#F2F4F6`(필드=회색, 패널=흰색의 **보더리스 스타일**) / r12 / h40 / 13.6px / placeholder `#8B95A1`
- 특수: datetime-local·date 네이티브, 금액 콤마 자동 포맷, disabled(인수증 시 수수료) 시각적 잠금

### 2-5. 패널/카드

- 패널: 흰 배경 / r18 / shadow-md / padding ~14×16
- 섹션 타이틀: 좌측 파랑 세로 바(`▎`) + 볼드 — "운행 등록", "운행 내역", "상세 정보" 공통 패턴
- 확장 행 내부 카드: 흰 배경 + 연한 보더로 중첩 위계

### 2-6. 테이블

- 헤더: `#8B95A1` 13px w700, 배경 투명 / 셀 padding 12×8
- 행: 구분선 최소화, hover `#F9FAFB`, 클릭 시 아코디언 확장
- 운임 컬럼: `#3182F6` w700 우측 정렬

### 2-7. 피드백

| 패턴 | 스타일 |
|---|---|
| 토스트 | 우상단 고정, 흰 배경 + 체크 아이콘 + shadow-lg ("운임 및 수수료 정보가 수정 완료되었습니다!") |
| 검증 | 네이티브 `alert()` — 누락 필드 나열 |
| 빈 상태 | 테이블 내 단일 행 안내문 |
| AI 어드바이저 배너 | 좌측 상태색 보더 + 상태색 연배경(`#E6F9F7` 등), 이모지 신호 + 지표 불릿 |

### 2-8. 레이아웃

- 사이드바 70px 아이콘 레일 (흰 배경, 우측 1px 보더, 활성 항목 `#E8F3FF` 배경)
- 본문 2패널: 좌 ~600px / 우 유동, 페이지 배경 `#F2F4F6` 위 흰 패널 부유
- 우측 패널 접힘 시: 우측 가장자리 파란 세로 텍스트 탭
- 반응형: ~390px에서 단일 컬럼, 테이블 3컬럼 축소, 필터 칩 일부 숨김

## 3. 시스템 관점 평가 (현황 요약)

**일관성 있는 부분**: 색·라운드·그림자·모션이 CSS 변수로 전부 토큰화되어 있고, 파랑=액션/주황=진행/청록=완료/빨강=취소의 시맨틱 매핑이 뱃지·버튼·어드바이저·신호등까지 일관 적용됨.

**정리 필요 (as-is 한계)**:
1. 폰트 스케일 8단계+ 소수 px — 토큰화 안 됨 (색상과 달리 변수 미사용)
2. 신호등 색(#10B981 등)이 `--success` 계열(#2EC4B6)과 **다른 초록** — 시맨틱 중복
3. 간격(spacing) 토큰 부재 — padding 값이 요소별 임의값
4. `--danger-text`(#DC2626)와 `--danger`(#F04452) 혼용
5. 성공 피드백(토스트)이 일부 액션에만 존재 — 피드백 규칙 미정의
