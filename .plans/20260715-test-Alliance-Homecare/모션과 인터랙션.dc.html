01 · 모션 원칙

모션은 돌봄의 태도를 닮아야 합니다

조용하고, 예측 가능하고, 재촉하지 않는 움직임. 화려함이 아니라 안심을 위한 모션입니다.

조용함

바운스·회전·번쩍임 금지. 이동은 24px 이하, 스케일은 1.05 이하.

목적

시선 유도와 상태 전달에만 사용합니다. 장식을 위한 모션은 없습니다.

일관성

같은 컴포넌트는 어느 페이지에서든 같은 시간·같은 곡선으로 움직입니다.

존중

모션 축소 설정 시 이동·스케일을 모두 끄고 120ms 페이드만 남깁니다.

03 · 카드

부상 + 이미지 줌, 두 겹의 깊이

전환 규칙
· 나가기: 콘텐츠만 opacity 0 · translateY(-12px) · 200ms — 배경색은 즉시 유지해 화면 번쩍임 방지.
· 들어오기: 히어로의 아이브로우 → 제목 → 본문 → CTA 순서로 70ms 스태거, 각 500ms enter.
· 브라우저가 지원하면 View Transitions API 크로스페이드 240ms를 우선 사용, 미지원 시 즉시 전환으로 폴백.
· 같은 페이지 내 앵커 이동은 smooth 스크롤 유지 + 각 섹션에 scroll-margin-top: 98px (고정 내비 높이 보정).
· 내비 액티브 밑줄은 페이지 진입 시 왼쪽에서 width 0 → 100%로 250ms에 그립니다.
