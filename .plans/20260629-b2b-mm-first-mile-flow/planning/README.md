# Planning 문서 목록

이 폴더는 B2B MM first-mile 수출입 flow를 업무 기획 산출물로 분해한 문서 묶음이다. Figma 보드는 운영 flow의 원천 자료이고, 이 폴더는 구현/상세 기획으로 넘기기 쉬운 구조화 문서다.

## 문서 순서

| 순서 | 파일 | 목적 |
| --- | --- | --- |
| 1 | `01-figma-source-analysis.md` | 원본 보드에서 직접 확인한 내용과 분석 한계 정리 |
| 2 | `02-process-prd.md` | 목표, 사용자, 범위, 요구사항, 수용 기준 정리 |
| 3 | `03-user-operation-flows.md` | 수출/수입/EDI/정산 업무 flow 정리 |
| 4 | `04-system-document-map.md` | PMS/TOMS/외부 시스템과 서류의 관계 정리 |
| 5 | `05-open-questions-and-assumptions.md` | 미결 질문, 가정, 의사결정 후보 정리 |
| 6 | `06-traceability-and-review.md` | Figma 근거와 문서 산출물의 추적성, self-review |

## 산출물 사용 기준

| 사용 상황 | 우선 참고 문서 |

| 데이터/상태/서류 모델링 | `04-system-document-map.md` |
| 다음 회의 질문 정리 | `05-open-questions-and-assumptions.md` |
| 근거와 리스크 확인 | `01-figma-source-analysis.md`, `06-traceability-and-review.md` |
