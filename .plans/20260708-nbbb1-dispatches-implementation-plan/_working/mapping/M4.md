## 관찰 메모

1. 대화방은 M4 담당 섹션 전체가 "자산 없음" 도메인이다 — Phase 3에서 소통 기능은 의도적으로 다루지 않았고(기각 기록조차 없음, 애초 범위 밖), 유일한 소통성 자산인 LMS는 legacy 상세 시트에 격리돼 있다.
2. 신규 기획 시 재사용 가능한 것은 UI 패턴이 아니라 주변 계약이다: 콘솔 store 패턴(D7-013), 화주/차주 연락처 snapshot(D1-001/D5-001), route-local design scope(D8-002).
3. 대화방 백엔드(메시지 저장·실시간 전달·읽음 처리)는 Phase 3의 "backend 무변경" 원칙(F7 memo, CargoMan 연동 모두 기존 서버 재사용) 밖의 첫 신규 서버 도메인이 될 가능성이 높다 — 별도 기획 패키지 규모로 산정 필요.
