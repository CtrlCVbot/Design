## 패키지 최종 상태 판정

이 패키지(F1 Workspace Layout Shell)는 **draft 상태의 계획 전용 패키지**다. `00-context/00-index.md`의 상태 값이 `draft`이고, `02-package/10-release-checklist.md`의 모든 항목(Scope/Verification/Review)이 `todo`다. 이 패키지 문서 내에서 구현 완료를 선언한 항목은 없다. 따라서 F1 신규 기능은 전부 '계획만'으로 기록한다. 단, 문서가 기존 구현으로 전제하는 컴포넌트(`BrokerOrderListMain`, `BrokerOrderPanelShell`)는 '완료(기존 구현, F1 이전)'로 별도 기록했다.
