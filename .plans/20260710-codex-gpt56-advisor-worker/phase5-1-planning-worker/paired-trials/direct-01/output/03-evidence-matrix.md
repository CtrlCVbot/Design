# Evidence Matrix

아래 매트릭스는 모든 material claim의 추적 경로를 제공한다. [REQ-01]

| Evidence ID | 기획 문서에서의 사용 |
|---|---|
| PROD-01 | 단일 패키지에서 결정과 범위를 확인하는 목적 |
| PROD-02 | 정확히 5개 문서의 Pilot 산출물 |
| REQ-01 | material claim의 evidence ID 추적 |
| REQ-02 | 상대 링크 유효성 |
| ARCH-01 | Option A Big-Bang rollout의 넓은 복구 범위 |
| ARCH-02 | Option B의 3단계 Gate 및 rollback 가능성 |
| DEC-01 | Option B 선택 |
| CON-01 | production/global config/source code 수정 금지 |
| RISK-01 | stale evidence 혼입 위험과 coverage 검증 대응 |
| RISK-02 | planning output 외 수정 위험과 hash 비교 대응 |
| GATE-01 | evidence coverage와 source hash 검증 |
| GATE-02 | 5개 문서와 내부 링크 검증 |
| GATE-03 | Advisor의 사실·추론·scope 독립 검증 |
