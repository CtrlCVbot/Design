# Pilot 범위

## 목적과 산출물

운영자는 여러 근거를 다시 탐색하지 않고 하나의 기획 패키지에서 결정과 범위를 확인해야 한다. [PROD-01] 이번 Pilot의 산출물은 정확히 5개 문서이며 코드 구현은 Non-Goal이다. [PROD-02]

## 요구사항

- 모든 material claim은 evidence ID로 추적 가능해야 한다. [REQ-01]
- 문서 간 상대 링크가 유효해야 한다. [REQ-02]

## 제약

production write, global configuration 변경, source code 수정은 금지한다. [CON-01]
