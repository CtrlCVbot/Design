# Scope and constraints

## Product scope

- `PROD-01`: The planning package must let the operating party confirm the
  decision and scope without re-searching dispersed evidence.
- `PROD-02`: This pilot produces exactly five documents; code implementation is
  a non-goal.

## Traceability requirements

- `REQ-01`: Every material claim must be traceable to an evidence ID.
- `REQ-02`: Each document link must be valid.

## Constraint

- `CON-01`: Production writes, global-configuration changes, and source-code
  changes are prohibited.
