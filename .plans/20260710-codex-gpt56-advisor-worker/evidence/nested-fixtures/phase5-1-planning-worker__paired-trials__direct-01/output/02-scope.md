# Pilot 범위

## 목적

운영자는 여러 근거를 다시 탐색하지 않고 하나의 기획 패키지에서 결정과 범위를 확인해야 한다. [PROD-01]

## 포함 범위

- Pilot 산출물은 정확히 5개 문서다. [PROD-02]
- 모든 material claim은 evidence ID로 추적 가능해야 한다. [REQ-01]
- 문서 간 상대 링크는 유효해야 한다. [REQ-02]

## 제외 범위와 제약

production write, global configuration 변경, source code 수정은 금지한다. [CON-01]

## 연결

근거 전체 매핑은 [03-evidence-matrix.md](./03-evidence-matrix.md)에서 확인한다.
