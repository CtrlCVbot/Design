| 항목 | 결정 |

| 디자인 기준 | `화물 배차 앱 3차 (standalone).html` 원본 디자인 유지 |

## 현재 결정

하단 탭은 `내 배차` / `운행 내역` / `정산` 3개로 유지한다. 단, `정산` 탭은 건별 `정산/송금 상세`가 아니라 `SCR-008 정산 홈`으로 진입한다.

## 고정 결과

현재 퍼블리싱 산출물은 아래 기준으로 고정한다.

| 항목 | 결정 | 근거 |
| --- | --- | --- |
| 하단 탭 | `내 배차` / `운행 내역` / `정산` 유지 | A안 기준으로 이미 기획 반영됨 |
| 정산 홈 첫 카드 | `이번 달 송금 요약` | 차주가 가장 먼저 볼 금액 기준 |
| 미정산 집계 문구 | `운행완료/정산대기 기준` | `하차완료`를 정산 가능 상태로 오해하지 않게 하기 위함 |
