# Cargo Order Admin HiFi Vercel Preview

This package deploys the cargo order admin HiFi master HTML as a standalone static Vercel preview.

## Source Of Truth

The canonical source file is:

```txt
C:\Work\Dev\Design\.plans\20260430-cargo-order-admin\wireframes\final-handoff\baseline\html\cargo-order-admin-hifi-master.html
```

`index.html` is a deploy copy. Do not edit it directly.

`favicon.ico` is a tiny placeholder file used to avoid browser 404 noise during preview checks.

## Sync

Run this from `C:\Work\Dev\Design`:

```sh
node apps/cargo-order-admin-hifi/scripts/sync-from-source.mjs
```

The script copies the canonical master HTML to:

```txt
C:\Work\Dev\Design\apps\cargo-order-admin-hifi\index.html
```

It also verifies that source and target SHA-256 hashes match.

## Local Preview

```sh
python -m http.server 8766 --bind 127.0.0.1 --directory apps/cargo-order-admin-hifi
```

Open:

```txt
http://127.0.0.1:8766/
```

## Vercel Preview Deploy

Preview deploy:

```sh
vercel deploy --target=preview --yes
```

Production deploy should be done only after review:

```sh
vercel deploy --prod --yes
```

## Access Policy

The deployment sets `X-Robots-Tag: noindex` and disables long-lived HTML caching.

For internal sharing, also enable Vercel Deployment Protection in the project dashboard.
